# RecipeHelper

一个FFXIV配方计算器

- **Github地址:** <https://paomiantong.github.io/RecipeHelper/#/>
- **Gitee地址:** <https://a8kt.gitee.io/recipe-helper/#/>

## Features

- [x] FFXIV 6.4
- [x] 统计所需军票、双色宝石、神典石、能工巧匠票据数量
- [x] 显示可采集材料的采集位置
- [x] 可添加已有材料数量，动态计算所需材料数量
- [x] 支持配方计算项目管理
